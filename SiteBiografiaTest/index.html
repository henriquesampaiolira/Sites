Sobre mim

Moro em Feira de Santana mas posso atender clientes do mundo todo. Faço produtos de qualidade, durabilidade e que agregam valor para meus clientes.

Trabalho usando a web como plataforma, de início utilizo HTML5 e CSS3 para criação de sites, sou um estudante que está sempre em desenvolvimento. Crio sites para todos, seguindo as principais diretivas de acessibilidade, responsividade e web semântica, além foco na qualidade de código.

Meu trabalho

O meu processo de desenvolvimento de sites tem foco em satisfazer o cliente com contato direto com o mesmo. Desta forma o cliente está sempre atualizado em como está o andamento do site.

Trabalho com escopo fechado e aberto, com preferência do cliente. Priorizando sempre a satisfação do cliente e qualidade do código.

Tecnologias

Estou construindo uma base sólida do HTML e CSS antes de partir para o JavaScript.

Tenho conhecimento do HTML semântico, que vai ajudar a alavancar seu site, ou seja, ser mais fácil de ser encontrado.

A minha relação com o CSS permite uma estilização de site única, na qual estou sempre me aperfeiçoando para entregar visuais modernos e práticos.
